Administração

Atribuir perfil de acesso a um usuário já criado no Firebase Auth

Crie a conta de login (e-mail/senha) direto no Console do Firebase → Authentication. Depois cole o UID aqui para definir nome e perfil de acesso dessa conta dentro do sistema. Se o perfil for Motorista, escolha também qual motorista cadastrado é essa pessoa — é isso que faz o login dela só abrir o veículo vinculado a ele.
